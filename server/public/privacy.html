Phantom Auth0 Privacy Policy

Phantom Auth0 is a separate security-focused build of Phantom designed around hosted authorization. The local browser extension remains responsible for page context and local interaction, while the companion surface handles identity, connected accounts, and approval-state visibility.

If you attach third-party accounts through Auth0, delegated access is mediated by the configured Auth0 tenant, its Connected Accounts setup, and Token Vault-backed runtime exchange. This hosted surface is intended to make delegated authority visible rather than hide it behind the agent runtime.

This public deployment is intended for evaluation and demonstration. Questions about this isolated build should be directed to the project repository: github.com/youneslaaroussi/phantom-auth0.
