Phantom Auth0 Privacy Policy

Phantom Auth0 is an isolated hackathon build. The local browser extension continues to keep page-reading, memory, and privacy-shield behavior on-device, while the hosted companion app handles sign-in, connected accounts, and approval state.

If you connect third-party accounts through Auth0, delegated access is mediated through the configured Auth0 tenant and its Connected Accounts / Token Vault setup. This repo does not intentionally write to the original Phantom deployment or public services.

Questions about this isolated build should be directed to the project repo: github.com/youneslaaroussi/phantom-auth0.
