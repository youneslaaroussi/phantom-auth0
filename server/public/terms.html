Phantom Auth0 Terms

This project is a separate Auth0-centered derivative of Phantom intended for evaluation, experimentation, and demonstration of delegated authorization flows for browser agents.

Using connected-account features in this project requires appropriate provider credentials, Auth0 tenant configuration, and authorization from the relevant external systems. High-risk action flows are designed to surface approval requirements before external state changes occur.

The source for this isolated build is available at github.com/youneslaaroussi/phantom-auth0.
