Phantom Auth0 Build Notes

This isolated repo exists for the Auth0 for AI Agents hackathon. It intentionally separates the original Phantom project from the new connected-account, Token Vault, and approval-driven action layer.

The companion app and action gateway were added during the hackathon submission period to demonstrate:

  • Auth0 Universal Login for a hosted companion experience
  • Extension pairing for a restricted local agent
  • Connected Accounts management for Google and GitHub
  • Delegated actions that distinguish read, draft, and send/create/post behavior
  • Approval-oriented flows for sensitive actions

Open the companion app or review the source at github.com/youneslaaroussi/phantom-auth0.
